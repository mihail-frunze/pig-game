# Pig Game

A simple dice game built with JavaScript for two players.

## Live Demo
https://pig-game-mihailfrunze.netlify.app/

## GitHub
https://github.com/mihail-frunze/pig-game

## How it works

- The game has 2 players
- Players take turns rolling a dice
- Each roll adds to the current score
- If you roll a **1**, you lose your current score and the turn switches
- You can choose to **Hold** to save your points
- First player to reach **100 points** wins the game

## What I practised

- JavaScript game logic
- Working with multiple variables (scores, current score, active player)
- DOM manipulation
- Event handling
- Switching between players
- Updating UI dynamically
